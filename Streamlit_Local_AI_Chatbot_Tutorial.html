Streamlit Chatbot with Local AI Model (No API Key)

This tutorial shows how to build a real AI-powered chatbot using Streamlit and Hugging Face Transformers without any API key. The model runs locally on your machine.

Goal: Create a professional project folder, install dependencies, understand every line of the Python code, and run a Streamlit chatbot that uses a real language model (distilgpt2).

1. Project Folder Structure (Professional Layout)

Use the following structure so that your chatbot project looks clean and professional:

streamlit_chatbot_local_ai/ ├─ app.py # Main Streamlit chatbot application ├─ requirements.txt # Python dependencies for this project ├─ docs/ # Documentation, notes, screenshots │ └─ README.md # High-level project description └─ templates/ # (Optional) Future prompt templates or config files

1.1 Script to Create Folder Structure (Windows PowerShell)

Run this in PowerShell to generate the structure automatically:

# Create main folder New-Item -ItemType Directory -Path "streamlit_chatbot_local_ai" | Out-Null # Go inside the folder Set-Location "streamlit_chatbot_local_ai" # Create sub-folders New-Item -ItemType Directory -Path "docs" | Out-Null New-Item -ItemType Directory -Path "templates" | Out-Null # Create empty files New-Item -ItemType File -Path "app.py" | Out-Null New-Item -ItemType File -Path "requirements.txt" | Out-Null New-Item -ItemType File -Path "docs\README.md" | Out-Null

1.2 Script to Create Folder Structure (macOS / Linux Bash)

Run this in a terminal (bash):

mkdir -p streamlit_chatbot_local_ai/docs mkdir -p streamlit_chatbot_local_ai/templates cd streamlit_chatbot_local_ai touch app.py touch requirements.txt touch docs/README.md

1.3 Sample requirements.txt

Put this inside requirements.txt:

streamlit transformers torch

2. How to Execute This Program

2.1 Create and Activate Virtual Environment (Recommended)

On Windows (PowerShell):

cd path\to\streamlit_chatbot_local_ai python -m venv venv venv\Scripts\activate

On macOS / Linux:

cd /path/to/streamlit_chatbot_local_ai python3 -m venv venv source venv/bin/activate

2.2 Install Dependencies

pip install -r requirements.txt

2.3 Run the Streamlit Chatbot App

streamlit run app.py

Your browser will open at a URL similar to http://localhost:8501. Type a message into the chat box and click Send. The local AI model will generate a reply.

5. Quick Reference / Cheat Sheet

Important Concepts in This Tutorial
Concept Description Where Used
st.session_state Stores persistent data (chat history, settings) across reruns. Chat messages list and temperature value.
@st.cache_resource Caches heavy objects (like models) so they are loaded only once. load_model() function.
pipeline("text-generation") High-level API to use text generation models. generator = pipeline(...)
Prompt building Combine history + new user message into one string. build_prompt_from_history()
Chat bubbles Use st.markdown with HTML to style messages. Loop rendering messages in Section 6.
